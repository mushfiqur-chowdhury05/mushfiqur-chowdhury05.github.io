BorderBlaze

I developed a web-based tool that can evaluate boundary point segmentation algorithms on a LiDAR point cloud dataset of building rooftops. The tool I implemented evaluates three different boundary point segmentation algorithms: Neighborhood Based Approach, Concave Hull Algorithm and Delaunay Triangulation Based Algorithm.

Save The Date: A Party Planner

I developed a party planning web application which serves as a centralized hub, connecting various party service providers (caterers, decorators, vendors etc.) with potential customers. Additionally, we developed a recommendation system using K-Nearest Neighbor algorithm to suggest a complete plan for services to avail based on user inputs like date, budget, and location.
